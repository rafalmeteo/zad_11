# User Stories

- Jako **menedżer marketingu**, chcę zobaczyć wykres nastroju opinii klientów, aby szybciej reagować na negatywne komentarze.
- Jako **analityk danych**, chcę móc wczytać dane z pliku CSV, aby analizować recenzje z różnych źródeł.
- Jako **product owner**, chcę raport podsumowujący kluczowe wskaźniki, aby ocenić sukces nowej funkcji.
