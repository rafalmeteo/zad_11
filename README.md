# Customer Insights App

Aplikacja do analizy opinii klientów.  
Pozwala wczytywać dane z recenzji, analizować sentyment i wyświetlać wyniki na dashboardzie.

## Struktura projektu
- `data/` – przykładowe dane wejściowe
- `src/` – kod źródłowy aplikacji
- `docs/` – dokumentacja

## Jak uruchomić
```bash
pip install -r requirements.txt
python src/sentiment_analysis.py
```
